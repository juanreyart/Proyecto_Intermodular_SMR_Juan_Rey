# Sistemas Operativos Monopuesto - Galería de Arte Digital

Este repositorio documenta todo el proceso de despliegue, configuración y endurecimiento (*hardening*) del sistema operativo base para las estaciones de trabajo de la **Galería de Arte Digital**. El objetivo es proporcionar un entorno de trabajo profesional, seguro y eficiente, optimizado para la gestión de activos digitales pesados y la administración de la red local.

## Índice del Proyecto
1.  **Elección del Sistema:** Justificación técnica del uso de Windows 11 Pro para un entorno profesional.
2.  **Despliegue y Particionado:** Proceso de instalación y estructuración del almacenamiento.
3.  **Configuración Inicial:** Ajustes de red, nombre de equipo y optimización regional.
4.  **Gestión de Usuarios:** Definición de roles (Administrador vs. Recepción) y políticas de privilegios mínimos.
5.  **Software Necesario:** Instalación de herramientas de gestión, edición multimedia y comunicación.
6.  **Seguridad Avanzada:** Protocolos de protección activa (Firewall, Antivirus y cifrado).

---

## Arquitectura del Sistema
El núcleo operativo de la galería se ha construido sobre **Windows 11 Pro**, seleccionando este sistema por su compatibilidad nativa con hardware de alto rendimiento y sus capacidades de virtualización y gestión de seguridad empresarial. El entorno está diseñado para aislar los datos críticos y permitir un mantenimiento preventivo sencillo.

## Especificaciones Técnicas y Seguridad
* **Gestión de Privilegios:** Implementación estricta de cuentas de usuario diferenciadas para evitar instalaciones no autorizadas.
* **Seguridad proactiva:** Uso de herramientas de aislamiento como Windows Sandbox para la auditoría de archivos recibidos y cifrado de disco para proteger datos sensibles.
* **Protección de Red:** Firewall configurado para bloquear accesos no solicitados, permitiendo únicamente el tráfico necesario para la operatividad de la galería.
* **Integridad:** Implementación de antivirus en tiempo real para asegurar que toda obra digital entrante esté libre de código malicioso.

## Resultados y Validación
* **Entorno Seguro:** El sistema operativo cumple con los estándares básicos de seguridad para evitar vectores de ataque comunes en entornos SMR.
* **Orden Administrativo:** La gestión de usuarios asegura que el personal de recepción solo tenga acceso a los recursos necesarios para su función, minimizando el riesgo operativo.
* **Continuidad:** La configuración de actualizaciones y controladores asegura que la estación de trabajo se mantenga estable y actualizada ante nuevas amenazas.
