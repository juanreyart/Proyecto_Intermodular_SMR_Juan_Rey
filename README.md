# Proyecto Intermodular SMR - Galería de Arte Digital

Este repositorio contiene el desarrollo integral de la infraestructura tecnológica para una **Galería de Arte Digital**. El proyecto aplica los conocimientos adquiridos en el primer curso del Ciclo Formativo de Grado Medio en **Sistemas Microinformáticos y Redes (SMR)**.

---

## 🚀 Descripción del Proyecto
La galería es un espacio híbrido diseñado para que artistas digitales expongan sus obras en pantallas de alta resolución. El sistema está optimizado para manejar archivos multimedia pesados con total fluidez, seguridad y una red de alta velocidad.

---

## 📂 Estructura del Repositorio
El proyecto se organiza según los módulos del ciclo:

* **[📁 Montaje y Mantenimiento](./Montaje_y_Mantenimiento/):** Inventario de hardware (Workstation i7, GPU 16GB), protocolos de prevención de riesgos, protección ambiental y plan de copias de seguridad.
* **[📁 Redes Locales](./Redes_Locales/):** Infraestructura de conectividad, incluyendo Switch de 16 puertos, Punto de Acceso Wi-Fi 6 y esquema de red.
* **[📁 Base de Datos](./Base_de_Datos/):** Diagrama Entidad-Relación (E/R) para la gestión de ilustradores, obras y clientes.
* **[📁 Itinerario Personal para la Empleabilidad](./Itinerario_Personal_Empleabilidad/):** Perfil profesional y análisis de empresas del sector en Galicia
* **[📁 Sistemas Operativos Monopuesto](./Sistemas_Operativos_Monopuesto/):** Configuración y optimización del sistema operativo para el puesto de edición.
* **[📁 Aplicaciones Ofimáticas](./Aplicaciones_Ofimaticas/):** Documentación y gestión administrativa de la galería.

---

## 🛠️ Especificaciones Técnicas Destacadas
* **Estación de Trabajo:** Procesador Intel Core i7-14700KF y 32GB de RAM DDR5.
* **Capacidad Visual:** GPU con 16GB de VRAM para gestión de 8 pantallas simultáneas.
* **Red:** Arquitectura Gigabit Ethernet cableada para máxima estabilidad.
* **Seguridad Eléctrica:** Protección mediante Sistema de Alimentación Ininterrumpida (SAI).
